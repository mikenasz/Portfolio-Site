NB CARES

The NB CARES website was created to host a calendar of events to promote self suffiency for residents in New Britan, CT. Created with a team of 4, and partnered with the New Britan Housing Authority and ParaDYM academy.

Main Features
  • Calendar of events where residents can view events and filter by category.
  • Event form for organizations.
  • Login/Registration for staff members.
  • Admin page for staff members to monitor the calendar.

Technologies Used: HTML, CSS, Javascript, Firebase, NodeJS, Bootstrap
